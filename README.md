A simple, beautiful wheel spinner for picking random names.

## Features

- Add 2-30 names
- Smooth spinning animation with realistic deceleration
- Colorful wheel segments
- Confetti celebration on winner
- Fully responsive (works on mobile)
- No dependencies - single HTML file

## Usage

Just open `index.html` in any browser. No server needed!

## Hosting

Drop the `index.html` file anywhere:
- GitHub Pages
- Netlify (drag & drop)
- Vercel
- Any static host

## Customization

Edit the `colors` array in the script to change wheel colors.
